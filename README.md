# LT TelopMover

テロップの位置・スケールを一括変更！

Premiere Pro のタイムライン上にあるグラフィッククリップ（テロップ）の **位置**・**アンカーポイント**・**スケール** をまとめて書き換える CEP 拡張機能です。

---

## ダウンロード

[最新版をダウンロード](../../releases/latest) — Assets から `.zxp` ファイルをダウンロードしてください。

---

## インストール方法

1. **ZXP インストーラーを用意**
   - [ZXP/UXP Installer](https://aescripts.com/learn/zxp-uxp-installer/)（推奨）
   - または [Anastasiy's Extension Manager](https://install.anastasiy.com/)

2. **`.zxp` ファイルをインストール**
   - インストーラーにドラッグ＆ドロップするだけで完了します。

3. **Premiere Pro で起動**
   - `ウィンドウ` > `エクステンション` > `LT TelopMover`

---

## 機能一覧

| # | 機能 | 説明 |
|---|------|------|
| 1 | 位置の一括変更 | 指定した X / Y 座標をまとめて適用 |
| 2 | アンカーポイントの一括変更 | アンカーポイント X / Y を一括設定 |
| 3 | スケールの一括変更 | 幅・高さを個別 or 縦横比固定で適用 |
| 4 | 対象範囲の指定 | 全クリップ / イン・アウト間 / 選択クリップ |
| 5 | プリセット保存・呼び出し | よく使う設定を名前付きで保存 |
| 6 | 実行前の確認ダイアログ | 誤操作を防ぐ二段階認証 |
| 7 | リアルタイムログ表示 | 処理状況をパネル内に表示 |

## 使い方

### 対象設定

| 項目 | 説明 |
|------|------|
| **Track (V)** | 操作対象のビデオトラック番号（V1 = 1, V2 = 2 ...） |
| **Range** | `All Clips`：トラック全体 / `In / Out Range`：イン〜アウト間 / `Selected Clips`：選択中のクリップのみ |

### トランスフォーム設定

| 項目 | 説明 |
|------|------|
| **Position** | チェックを入れると X / Y 座標を指定値に書き換え（ピクセル単位） |
| **Anchor Point** | チェックを入れるとアンカーポイント X / Y を指定値に書き換え |
| **Scale** | チェックを入れるとスケールを書き換え。Uniform ON で縦横比固定 |

### プリセット

| 操作 | 説明 |
|------|------|
| **Select Preset** | 保存済みプリセットを選んで即座に反映 |
| **Save** | 現在の設定を名前をつけて保存 |
| **Delete** | 選択中のプリセットを削除 |

### 実行

**Execute** ボタンを押すと確認ダイアログが表示されます。OK を押すと処理が実行され、ログエリアに結果が表示されます。

## 設定例

### 例 1：V2 テロップを画面中央に移動

| 設定項目 | 値 |
|---------|-----|
| Track | 2 |
| Range | All Clips |
| Position | X: 960 / Y: 540 |

### 例 2：V3 テロップのスケールを 80% に統一

| 設定項目 | 値 |
|---------|-----|
| Track | 3 |
| Range | Selected Clips |
| Scale | W: 80 / Uniform: ON |

---

## 注意事項

- 操作対象のトラックが **ロックされていないこと** を確認してください。
- 実行前にプロジェクトを **保存** しておくことを推奨します。
- 書き換え対象は **クリップ全体のモーション（Motion / Vector Motion）** プロパティです。エッセンシャルグラフィックス内の個別テキストレイヤーには影響しません。
